# Selenium Python Automation Template

This is a starter automation framework using:
- Python - 
- Selenium
- Pytest
- WebDriver Manager

## Setup Instructions

1. Clone the repository - git clone https://github.com/unaisLearning/selenium-python-template.git

2. Create virtual environment:
   python3 -m venv venv
3. Activate venv:
   source venv/bin/activate
4. Install dependencies:
   pip install -r requirements.txt

## Run Tests

pytest -v

## Folder Structure

- tests/  -> test cases
- pages/  -> page objects
- utils/  -> utilities
